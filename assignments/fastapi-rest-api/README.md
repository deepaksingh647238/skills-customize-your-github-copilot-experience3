# 📘 Assignment: Building REST APIs with FastAPI

## 🎯 Objective
Learn how to build and deploy RESTful APIs using the FastAPI framework in Python. You will practice designing endpoints, handling requests, and returning responses, while building a simple API for managing resources.

## 📝 Tasks

### 🛠️ Set Up FastAPI Project
#### Description
Initialize a FastAPI project and install necessary dependencies.
#### Requirements
Completed program should:
- Have FastAPI and Uvicorn installed
- Contain a main Python file to run the API

### 🛠️ Create Basic Endpoints
#### Description
Design endpoints for basic CRUD operations (Create, Read, Update, Delete) for a resource (e.g., books, students, tasks).
#### Requirements
Completed program should:
- Implement GET, POST, PUT, DELETE endpoints
- Handle JSON request and response bodies
- Return appropriate status codes

### 🛠️ Test and Run the API
#### Description
Test your API using a tool like curl or Postman, and run it locally with Uvicorn.
#### Requirements
Completed program should:
- Start the API server with Uvicorn
- Demonstrate successful requests and responses for each endpoint
- Include example test cases or curl commands in README
